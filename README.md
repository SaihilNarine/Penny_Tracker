# PennyTracker App

## Description

PennyTracker is a budget tracking Android application that allows users to:

* Add expenses
* Set monthly goals
* View spending insights
* Analyse data using graphs
* Unlock Achievements

## Features

* User login system
* Expense tracking
* Category management
* Budget limits (min & max goals)
* Graph visualisation
* Offline storage using Room Database
* Bottom Navigation (Added Feature)
* Achievements (Added Feature)

## Technologies Used

* Kotlin
* Android Studio
* Room Database
* Material UI
* Firebase database
