# PennyTracker App

## Description

Penny Pilot is a free Android budgeting application developed to help users manage their expenses and financial activities responsibly.
Penny Pilot is a budget tracking Android application that allows users to:

* Add expenses
* Set monthly goals
* View spending insights
* Analyse data using graphs
* Unlock Achievements
* Assist users in tracking daily expenses and categorizing them for better financial awareness
* Offer a convenient, accessible, and user-friendly platform for personal finance management

## Features
* User login system
* Expense tracking
* Category management
* Budget limits (min & max goals)
* Graph visualisation
* Offline storage using Room Database
* Bottom Navigation (Added Feature)
* Achievements (Added Feature)

## Technologies Used
* Kotlin
* Android Studio
* Room Database
* Material UI
* Firebase database
* GitHub
* GitHub Actions

## Considerations
The design of Penny Pilot is centered on usability, security, and effective financial management.
Penny Pilot is a budget tracking Android application that takes inconsideration of:
* Simplicity and Intuitiveness
* Responsive Layout
* Easy Navigation
* Sensitive Data Protection
* Visual Aids

Penny Pilot is designed to be an effective, user-friendly tool for personal finance management. Its thoughtful design considerations ensure that users can easily track, analyze, and improve their financial habits while enjoying a secure and engaging experience.
